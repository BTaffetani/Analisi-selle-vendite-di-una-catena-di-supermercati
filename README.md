# Analisi delle vendite di una catena di supermercati

## Descrizione del Progetto

L'obiettivo del progetto era la creazione di un report visivo sulle performance della divisione europea di una catena di supermercati. In particolare, dopo aver creato una dashboard che riassumesse i dati di vendita e profitto nel mercato europeo, si chiedeva di creare una presentazione (story) allo scopo di guidare la decisione di allocare il budget destinato al marketing.

## Il Dataset

Il dataset fornito consiste in un file Excel contenente i dati degli ordini del mercato europeo.

## Struttura del Repository

Il progetto è organizzato come segue:

*  `notebooks/`: Il Tableau Workbook in cui sono raccolte le fasi del progetto.
*  `data/`: Il file Excel contenente i dati per il progetto.
*  `README.md`: Questa guida descrittiva.

## Metodologia e Fasi del Lavoro

Mentre per il primo obiettivo è stata creata un'unica dashboard, per il secondo sono state create due stories distinte. Nella prima l'approccio è stato più generico sul mercato europeo, mentre nella seconda ho mantenuto il focus sui Paesi Bassi, ossia il mercato meno profittevole secondo i dati forniti.
